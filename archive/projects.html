Created an app in Android Studio, to load photos from a users device where they can create albums and add custom tags to there photos.

  • Add tags to photos and search for photos by tags

HTTP Server

Developed a multi-threaded HTTP server from scatch in Java that accepts GET, POST, and HEAD requests from clients and responds with a well-formed HTTP response. The server is also capable of running CGI scripts with optional parameters from post requests

What I learned

  • Learned the Hypertext-Transfer-Protocol in depth
  • Working with sockets in Java
